图片隐写整理

MISC
发布日期:   2020-05-24

图片隐写分析

[toc]

文件格式

PNG文件

PNG文件头为89 50 4E 47 0D 0A 1A 0A

主要数据块包括IHDR IDAT IEND

IHDR

00 00 00 0D 说明IHDR头块长为13

49 48 44 52 IHDR标识

00 00 00 08 图像的宽

00 00 00 08 图像的高

······

36 21 A3 B8 CRC校验

IDAT

一个PNG文件可能有多个IDAT区块，IDAT区块是经过压缩的，可用zlib库查看

00 00 00 27 数据长

49 44 41 54 IDAT标识

IEND

通常为00000000 49454E44 AE426082

考点

  • 文件头修复

  • IHDR控制宽度和高度，用CRC32脚本爆破宽高，在使用winhex等工具修改

  • IDAT隐藏信息，tweakpng发现异常IDAT，在winhex中提取出来用zlib压缩得到文件后再进行相应处理

    import zlib

import binascii

IDAT = "789C7D913D48C3401CC55F53A5452A0E661071C8505DB4202AEA28552C8285D25668D5C1E4D22F68D290A4B8380AAE05073F16AB0E2ECEBA3AB80A82E0078893A393A28B94F8BFA4D022D683E37EBCBBF7B87B0708F532D3ACAE7140D36D33198B4A99ECAA147885001141CC60546696114F2DA6D1717CDDC3C7D7BB08CFEA7CEECFD1ABE62C06F824E239669836F106F1F4A66D70DE271659515689CF89C74CBA20F123D7158FDF38175C".decode('hex')

result = binascii.hexlify(zlib.decompress(IDAT))

print (result.decode('hex'))

print (len(result.decode('hex')))

JPG文件

文件头 FFD8FF

文件尾FFD9FF

其中FFC0(段标记)0011(数据长度)08(精度)03CF(图像高度)02BC(图像宽度)

考点

  • 文件头修复
  • 图片高度修改

其他

GIF文件头47 49 46 38

PSD文件头38 42 50 53

常见解题

常用工具

stegsolve、tweakpng、python、winhex、binwalk、foremost、PS

常见解法

初级

  1. 若无法打开文件，用winhex打开后，查看文件头是否破损或者缺失进行补全

  2. winhex打开文件，Crtl+F查找文本信息如flag

  3. 查看图片属性，寻找是否有提示信息

  4. stegsolve打开查看各个通道，会有二维码、flag信息等

  5. binwalk/foremost 分离图片隐藏文件

  6. GIF文件处理

    1. 用gifsplitter软件分离

    2. 用convert命令分离

      convert test.gif flag.png

    3. 用python脚本分离

      from PIL import Image

gif = Image.open("test.gif")
for i,frame in enumerate(ImageSequence.Iterator(gif),1):
  if frame.mode == "JPEG":
        frame.save("%d.jpg"%i)
  else:
        frame.save("%d.png"%i)

中级

  1. LSB隐写

    使用stegsolve打开Analyse->Data Extract,查看各个通道的信息。

  2. 盲水印

    当有两张一模一样的图片，但文件大小不同时，可以用脚本发现得到其中的盲水印

  3. Image Combiner

    当有两张宽高相同的图片时可以使用stegsolve中Image Combiner

  4. 二维码

    1. QR_Research读取

    2. 二维码顶点补全以及二维码反色(PS打开后Crtl+I)

    3. 二维码模糊时，可用ps进行清晰度等等的调整。或查看可否在文件信息中找到原图，再在stegsolve中进行Image Combiner

    4. 当获取得到01信息时，用脚本形成二维码。

      import PIL
from PIL import Image
MAX = 260
# MAX=二维码每边的像素个数
img = Image.new("RGB",(MAX,MAX))
str="0101·······"
i = 0
for y in range (0,MAX):
    for x in range (0,MAX):
        if(str[i] == '1'):
            img.putpixel([x,y],(0, 0, 0))
        else:
            img.putpixel([x,y],(255,255,255))
        i = i+1
img.show()
img.save("flag.png")

  5. 图片拼接

    1. 少量的图片拼接可使用ps，或者ppt
    2. 大量细微的图片拼接，可采用linux中的指令或者自行写脚本
    montage flag*.png -tile x1 -geometry +0+0 flag.png

    -tile 是拼接时每行每列的图片数，x1即1行

    -geometry是首选每个图和边框尺寸，这里边框为0

  6. 部分图片文件可直接用7-zip打开
