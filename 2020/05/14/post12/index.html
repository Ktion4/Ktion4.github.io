0.写在前面

第一篇Blog，感觉写啥都可以。挑了几道有意思的题。

1.再见李华

打开压缩包后得到图片，发现上面有一串md5值，但是会发现只有17位，正常MD5为20位，猜测需要进行爆破。

将图片binwalk之后得到一个加密的zip包。根据比赛的提示

假如你是李华（LiHua)，收到乔帮主一封密信，没有任何特殊字符，请输入密码，不少于1000个字。同学，记得署名哦～

首先没有任何特殊字符；其次不少于1000个字，密码不可能这么长，猜测为2进制，1000=8，得密码可能为9位。最后记得署名,猜测密码最后5位为LiHua，密码还差前4位。所以编写python脚本爆破密码。

from hashlib import md5
import string

dic = string.printable

half_md5 = "1a4fb3fb5ee12307"

str1 = ""
for i1 in dic:
    for i2 in dic:
        for i3 in dic:
            for i4 in dic:
                str1 = i1 + i2 + i3 + i4 + "LiHua"
                str1_md5 = md5(str1.encode('utf-8')).hexdigest()
                if half_md5 in str1_md5:
                    print(str1)
                    exit(0)

得到密码为

15CCLiHua

打开压缩包,得到key.txt

Stay hungry, Stay foolish.

flag无格式，直接提交即可。

2.就在其中

法1

解压得到流量包，binwalk得到乱码的flag.txt，在binwalk结果的最前面看到RSA private key.

dd指令提取私钥相关内容保存为psa.key文件(这个还不太会用，分离出的内容多了很多，cat读取后将有用的部分再重新另外保存为psa1.key，方法有点笨啊啊啊啊)

dd if=Misc-03.pcapng of=psa.key bs=1 skip=44150 count=13036

用openssl解密

openssl rsautl decrypt -in key.txt -inkey psa1.key -out flag.txt

打开flag.txt就可以看到flag啦！

法2

解压得到流量包，wireshark追踪了几个tcp流发现了一些文件。于是用NetworkMiner分离得到key.txt和test.key等文件，其中test.key为私钥。

注意:NetworkMiner不可直接打开pcapng文件，先用wireshark另存为pcap文件，再用NetworkMiner打开。

至此可以使用法1的openssl解密，也可进入以下网站在线解密。

rsa在线解密

注意:key.txt需先转换成base64的形式再上传，转换可以用脚本。

import  base64
fin = open("key.txt", "r")
fout = open("flag.txt", "wb")
base64.encode(fin, fout)
fin.close()
fout.close()

3.奇怪的TTL字段

打开压缩包后，发现ttl.txt文件中只由63,127,191,255四个值组成。通过63,127,255猜测可能与2进制相关。写出这4个值得二进制后发现其后六位相同，前两位分别为00,01,10,11。从而猜测每四个可得一个asicc码，手动转换了前几组发现为FFD8，猜测可能会是jpg文件，运用如下脚本转换。

fp = open('ttl.txt','r')
a = fp.readlines()
p = []
for i in a:
    p.append(int(i[4:]))
s = ''
for i in p:
    if i == 63:
        a = '00'
    elif i == 127:
        a = '01'
    elif i == 191:
        a = '10'
    elif i == 255:
        a = '11'
    s += a

import binascii
flag = ''
for i in range(0,len(s),8):
    flag += chr(int(s[i:i+8],2))
flag = binascii.unhexlify(flag)
wp = open('1.jpg','wb')
wp.write(flag)
wp.close()

发现只有二维码的一部分，binwalk发现有6张图片，但是不知道为什么没有分离出来，于是用winhex打开逐个分离出来，再拼接成一个完整的二维码（可在ppt中拼）。扫描得到以下内容。

key:AutomaticKey cipher:fftu{2028mb39927wn1f96o6e12z03j58002p}

猜测维吉密码，进入维吉在线,需要选取AutoKey Variant哦

flag{2028ab39927df1d96e6a12b03e58002e}

4.miscmisc(脑洞题)

1)binwalk图片得到一张图和一个加密的压缩包

2)binwalk得到的图片(chayidian.jpg)，得到flag.txt发现与加密压缩包中的flag.txt的crc32值相同。进行明文攻击得到密码为z$^58a4w

3)解压后得到一个加密压缩包，一个word，一个png图(我快崩溃了。。)

4)png文件通过LSB BGR得到pass:z^ea,在word文件中看到隐藏文字。看了别人的wp，得知密码为pass内容+隐藏文字每行的最后一个字符,密码为z^ea4zaa3azf8。打开压缩包后得到flag。

这脑洞太太太大了(最后一个密码谁能想到啊啊啊啊啊啊啊啊啊啊)。

flag{12sad7eaf46a84fe9q4fasf48e6q4f6as4f864q9e48f9q4fa6sf6f48}

5.心仪的公司

流量包过滤http流,发现两个内网ip，筛选其中一个ip，发现有一个含jpg的TCP流，即可在末尾找到flag。

fl4g:{ftop_Is_Waiting_4_y}

6.双色块

binwalk gif得到gif和png，png中的内容为：key：ctfer2333

将gif分离得到576桢图片，绿色为0,紫色为1，得到的二进制转换成ascii码，再进行DES解码得到flag。

这题不咋会写，就单纯记录一下。
